# Password Generator

Простое приложение с графическим интерфейсом для генерации случайных паролей. Позволяет настраивать длину, количество паролей и набор символов (строчные, прописные, цифры, специальные символы). Результат можно копировать в буфер обмена.

![Скриншот](screenshot.png)

## Возможности

- Генерация одного или нескольких паролей (до 100)
- Настройка длины пароля (до 1000 символов)
- Выбор используемых наборов символов:
  - Строчные латинские буквы (a–z)
  - Прописные латинские буквы (A–Z)
  - Цифры (0–9)
  - Специальные символы (!@#$)
- Защита от редактирования результата, но возможность выделения и копирования (правой кнопкой мыши или Ctrl+C)
- Сброс всех настроек к значениям по умолчанию (длина 14, 1 пароль, все наборы включены)
- Клавиша Enter для быстрой генерации

## Требования

- **Python** версии 3.6 или выше
- Стандартная библиотека Python (tkinter и os входят в поставку)

## Установка и запуск

### Запуск скрипта

1. Убедитесь, что Python установлен и добавлен в `PATH`.
2. Дважды кликните на файл
