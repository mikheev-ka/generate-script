## Требования

- **Python 3.6 или выше** (только если запускать из исходного кода).
- Дополнительные библиотеки не требуются – используется только стандартная библиотека (`tkinter` входит в состав Python).


Как пользоваться
1. Укажите длину пароля (по умолчанию 14).

2. Укажите количество паролей (по умолчанию 1).

3. Выберите набор символов, отметив нужные чекбоксы. По умолчанию выбраны все.

4. Нажмите кнопку Сгенерировать (или клавишу Enter).
В поле результата появятся сгенерированные пароли.

5. Чтобы скопировать результат, выделите текст и нажмите Ctrl+C или щёлкните правой кнопкой мыши и выберите Копировать.

6. Кнопка Сбросить возвращает все настройки к исходным и очищает поле результата.
